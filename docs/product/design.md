# Focusly - Validation Screen Inventory

**Date:** 24 May 2026  
**Owner:** [STRATEGY.md](../STRATEGY.md)  
**Purpose:** Define only the surfaces needed for validation.

---

## Required Validation Surfaces

### 1. Fake-door landing page

Purpose: measure demand before building.

Sections:

- headline,
- problem statement,
- three-step "how it works",
- privacy block,
- competitor-aware positioning,
- beta CTA,
- optional paid-intent CTA.

Form fields:

- email,
- role,
- Mac primary device yes/no,
- biggest pain,
- willingness to pay,
- interview opt-in.

### 2. Prototype task capture

Purpose: test first-session friction.

Requirements:

- one task input,
- no account required,
- start session in under 10 seconds,
- no AI or task breakdown.

### 3. Always-visible focus anchor

Purpose: test whether the core Mac surface is useful.

Requirements:

- shows task,
- shows elapsed or remaining time,
- stays visible while user works,
- can be dismissed or paused.

### 4. Drift nudge toast

Purpose: test the key differentiator.

Requirements:

- gentle copy,
- manual trigger acceptable for prototype,
- optional heuristic trigger if code exists,
- buttons: "Back to task", "Snooze", "Pause nudges".

### 5. Re-entry note

Purpose: test whether users value session continuity.

Requirements:

- appears at session end,
- asks "Where should future you resume?",
- stores locally or in prototype state.

---

## Explicitly Deferred Screens

Do not design/build these during validation:

- full marketing site,
- blog,
- pricing page beyond fake-door price test,
- App Store screenshots,
- Setapp assets,
- onboarding carousel,
- AI voice settings,
- body-double match screen,
- coach dashboard,
- iOS app,
- Watch complication,
- detailed analytics,
- referral page.

---

## Copy Constraints

Use:

- "ADHD-aware work",
- "focus anchor",
- "drift-aware nudge",
- "privacy-first Mac utility."

Avoid:

- "treat ADHD",
- "medical",
- "clinically proven",
- "coach replacement",
- "surveillance",
- "always-on copilot" unless interviews show that language resonates.
